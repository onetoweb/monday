===========
Basic Usage
===========

Setup client

.. code-block:: php
    
    require 'vendor/autoload.php';
    
    use Onetoweb\Monday\Client as MondayClient;
    
    // param
    $apiKey = 'api_key';
    
    // setup client
    $client = new MondayClient($apiKey);
